  const PageHome = () => {
    const article3Ref = useRef<HTMLElement | null>(null)
    const [article3Intersecting, setArticle3Intersecting] = useState(false)
  
    useTitle('Home')
  
    /* ======================
            useEffect()
    ====================== */
  
    useEffect(() => {
      if (article3Ref.current === null) { return }
  
      const observer = new IntersectionObserver(
        (entries, _observer) => {
          const entry = entries?.[0]
  
          if (entry) {

          }
        },
        {
          rootMargin: '0px',
          threshold: 0.5
        }

      observer.observe(article3Ref.current)

Using A Custom Hook:

For this Colby Fayock switched to react-intersection-observer (1,415,821 hits), not to be confused with react-intersection-observer-hook (75,951 hits). Initially, I used the latter, and it also worked well enough. Alteratively, if you want a very low kb solution that is less magical, you could go for useIntersectionObserver() from usehooks-ts.com

I created AnimationDemo that uses a local version of react-intersection-observer. The actual hook isn't that difficult to grasp once you're looking at it in a single file (or two). I also created AnimationDemo2 that uses a local/enhanced version of useIntersectionObserver(). It's enhancedments include skip, delay (Edge/Chrome only - no Safari/Firefox), and onChange, which make it more or less like react-intersection-observer. That said, I'd still go with react-intersection-observer (local or the npm package) since it's a little more advanced.

In any case, Now we can do away with useState, useEffect, and useRef and just do this: 


  import { useInView } from 'react-intersection-observer'

  // Custom imports
  import { useTitle } from 'hooks'
  import { HR, Title } from 'components'
  

  /* ========================================================================
                                  PageHome
  ======================================================================== */

  const PageHome = () => {
    useTitle('Home')
    const { ref: article3Ref, inView: article3Intersecting /* , entry */ } = useInView({ threshold: 0 })

    /* ======================
            return
    ====================== */

    return (
      <div className='relative mx-auto w-full flex-1 p-6 2xl:container'>...</div>
    )
  }

  export default PageHome
